Правила CSS:
    1) Правила подключения стилей прямо на странице:
        - Используется тег <style></style>
            <style>
                Правила стилей
            </style>
        - Данный кусок кода CSS должен распологаться перед закрывающим тегом </head>.
        - Правила обращения - обращения бывают следующих типов:
            1) Теги     p
            2) Классы   .
            3) id       #
            4) атрибуты []

    2) Правила написания стилей:
        - Начинается все с того, кому мы должны присвоить стили?
            class
            id
            tag
            attribute
        - Мы должны открыть и закрыть фигурную скобку 
            {...}

            Целостная картина равна:
            class {

            }

            id {

            }

            tag{

            }

            attribute {

            }
    
    3) Закрывающая фигурная скобка должна быть параллельна названию класса или же любого свойства.
