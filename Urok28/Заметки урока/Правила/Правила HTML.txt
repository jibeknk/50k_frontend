Правила HTML:
    0) Курсор должен находиться между открытым и закрытым тегом.
    0) После того как мы создадим тег нажать на ENTER.
    1) Правила Tab'ов
        - Прежде чем писать код либо содержимое, нужно убедиться что в начале открытого и закрытого тега есть один TAB.
    2) Каждый код должен находиться между открытым и закрытым тегом.
    3) Каждый тег должен начинаться с открывающей крышечкой "<" и заканчиваться с закрывающей крышечкой ">".
    4) Правила параллельности
        - Каждый открытый тег должен быть параллельным закрытому тегу. И наоборот, каждый закрытый тег должен быть параллельным открытому тегу.
        Например:
                <div>
                    содержимое
                    содержимое
                    содержимое
                </div>
    5)ВСЕГДА НАБЛЮДАЙТЕ ЗА АКТИВНОЙ ПОЛОСОЙ, КОТОРАЯ НАХОДИТЬСЯ СЛЕВА ОТ ТЕГА - НА НЕЙ НИЧЕГО НЕ ДОЛЖНО БЫТЬ.
        Например:
            <div>
                содержимое
                содержимое
                содержимое
            </div>
    6) Каждый ВНУТРЕННИЙ ТЕГ либо содержимое должен содержать 1 ТАB в право.
        Например:
            <div>
                содержимое
                содержимое
                содержимое
            </div>
    7) Каждый СОСЕДНИЙ ТЕГ не должен содержать дополнительный TAB в право.
        Например:
            <div>
                содержимое
                содержимое
                содержимое
            </div>
            <div>
                содержимое
                содержимое
                содержимое
            </div>
    8) Правило "Стрелочка скрытия кода"
        - Всегда пробуйте свернуть свой код. Если не получается его свернуть - ЗНАЧИТ СТРУКТУРА НЕ ПРАВИЛЬНАЯ.
        Например:
            <div>
                содержимое
        содержимое
                содержимое
            </div>

Правила Атрибутов:
    1) class="" - Каждый атрибут должен начинаться со СВОЙСТВА: CLASS / ID / DATA-USER
                - Далее пишется равно "=" без каких-либо пробелов!
                - Кавычки после равно "=" пишутся без пробелов!
                - Если кавычка начинается с двойной ", она обязательно должна заканчиваться двойной ".        
                - Если кавычка начинается с одинарной ', она обязательно должна заканчиваться одинарной '.
